## 审计期间实施的修正

1. M05 原世界宽度为 1536，在 1920 宽视口下会露出右侧清屏色。新增 `camera.fitViewportWidth` 与窗口缩放回调；1920×900 实测为 1.25× 相机缩放，可见世界宽度精确为 1536，不再露黑边。
2. M04 BOSS 原单张运行表边长过大，部分浏览器/GPU 会在纹理上传时重试后失败。现按三阶段拆为 3 张 `1280×1440` RGBA 图集，保留原生 `320×360` 单帧，运行时按动作自动切表。
3. BOSS 死亡动画偶发不回调时，闪烁消失可能拖到 3.8 秒以上。已将兜底启动点收紧到 2.1 秒，异常路径也能稳定完成倒地、闪烁和消失。
4. Windows 下浏览器取消视频 Range 请求会产生 `ConnectionAbortedError` 日志；服务端已将其作为正常客户端断开处理。
5. iPhone MOV 母片仅本地保留并加入 Git 忽略；线上使用的 2K30 H.264/AAC MP4 交由 Git LFS 管理。
