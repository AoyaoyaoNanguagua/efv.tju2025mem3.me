# EFV基础玩法框架

## 1. 基础定位

EFV是一款单人轻量 RPG。玩家从五位学术喵中选择一位角色，通过挂机成长、地图探索、自走战斗、道具收集和章节挑战推进主线。

本框架以“学分压制 + 校园币经济 + 增益道具收集”为核心，弱化复杂装备和复杂攻防公式，让玩家主要围绕学分、血量、暴击、道具搭配和章节挑战进行成长。

## 2. 核心循环

玩家基础循环为：

```text
打怪/自动寻路打怪/完成地图任务
→ 获得 EXP、校园币、材料或道具
→ 进入章节地图探索
→ 打普通怪、精英、稀有精英、Boss
→ 获得学分、校园币、材料、增益装备
→ 强化背包配置
→ 挑战章节 Boss
→ 进入下一章节
```

章节推进不再把学分作为绝对硬门槛。玩家学分不足时仍然可以挑战 Boss；如果成功击败 Boss，也可以进入下一章节。

## 3. 核心资源

| 资源 | 来源 | 用途 |
| --- | --- | --- |
| EXP | 挂机、战斗、事件、失败补偿 | 提升等级和基础属性 |
| 学分 | 地图探索、精英/稀有精英、Boss、部分事件 | 影响特定敌人的伤害倍率和章节挑战难度 |
| 校园币 | 实习、战斗、宝箱、出售道具、事件 | 开宝箱、购买道具、合成或交易 |

## 4. 学分机制

学分是章节挑战的核心数值，但不是硬性通关要求。

部分精英、稀有精英和 Boss 拥有“学分防御”机制。玩家当前学分越接近敌人的要求学分，对敌人造成的伤害越接近正常水平；学分不足时伤害降低。

### 4.1 玩家对学分防御敌人的伤害

```text
玩家伤害倍率 = 玩家当前学分 / 敌人要求学分
```

例子：

```text
玩家当前学分 = 5
Boss 要求学分 = 10

玩家对 Boss 造成伤害 = 原伤害 × 50%
```

建议在实际数值中设置上下限，避免过度惩罚或过度溢出：

```text
玩家伤害倍率 = clamp(玩家学分 / 敌人要求学分, 30%, 120%)
```

### 4.2 Boss 主动学分技能

Boss 可以拥有主动型学分技能。该类技能根据 Boss 要求学分与玩家当前学分比例提高伤害。

```text
Boss 学分技能伤害倍率 = Boss 要求学分 / 玩家当前学分
```

例子：

```text
玩家当前学分 = 5
Boss 要求学分 = 10

Boss 主动学分技能对玩家造成伤害 = 原伤害 × 200%
```

建议设置上限：

```text
Boss 学分技能倍率 = clamp(Boss 要求学分 / 玩家学分, 100%, 200%)
```

Boss 普通攻击和普通技能不受该机制影响，仍按正常伤害结算。

### 4.3 适用范围

| 敌人类型 | 是否默认拥有学分防御 |
| --- | --- |
| 普通怪 | 否 |
| 普通精英 | 通常否，可按关卡设计少量配置 |
| 稀有精英 | 可配置 |
| Boss | 默认配置 |
| 特殊剧情怪 | 按剧情需要配置 |

普通怪和大部分普通精英保持简单扣血，不受学分影响。

## 5. 战斗伤害框架

整体战斗不使用复杂攻防公式，基础逻辑以扣血为主。

基础流程：

```text
基础伤害
→ 学分倍率
→ 暴击倍率
→ 扣除目标生命值
```

暴击机制保留：

| 类型 | 暴击倍率 |
| --- | --- |
| 物理暴击 | 150% |
| 魔法暴击 | 200% |

示例：

```text
基础伤害 = 100
玩家学分不足，对 Boss 伤害倍率 = 50%
触发魔法暴击 = 200%

最终伤害 = 100 × 50% × 200% = 100
```

低学分玩家仍有机会通过等级、生命、暴击和增益装备完成越级挑战。

## 6. 校园币经济

校园币承担轻量经济系统功能。

主要用途：

| 用途 | 说明 |
| --- | --- |
| 开宝箱 | 消耗校园币获得材料、道具或增益装备 |
| 购买道具 | 购买消耗品、材料或基础增益道具 |
| 出售道具 | 材料和增益装备可出售换取校园币 |
| 合成消耗 | 后续可用于材料合成或道具合成 |

章节校园币可以用于当前章节宝箱和章节内消耗，形成“探索 - 获取校园币 - 开宝箱 - 获得装备/材料 - 继续挑战”的循环。

## 7. 道具分类

道具分为三类：

| 类型 | 说明 |
| --- | --- |
| 材料 | 可出售，可用于合成道具或任务提交 |
| 任务道具 | 剧情任务、章节任务或特殊事件所需 |
| 增益装备 | 放入装备栏，提供属性、收益或套装效果 |

道具采用校园风格命名和描述，例如课程资料、饭卡、门禁卡、书签、便签、讲座海报、实验材料等。

## 8. 增益装备系统

由于角色精灵图资源有限，装备不体现为外观变化，而转化为“增益装备”概念。

增益装备特点：

- 不改变角色外观。
- 放入装备栏后生效。
- 装备栏上限为 10 格。
- 可出售换取校园币。
- 可通过精英、稀有精英、Boss、宝箱掉落。

## 9. 装备品质

增益装备按品质分级，并使用对应描述颜色。

| 品质 | 颜色 | 说明 |
| --- | --- | --- |
| 普通 | 白 | 基础加成 |
| 优秀 | 绿 | 小幅加成 |
| 精良 | 蓝 | 稳定加成 |
| 史诗 | 紫 | 较强加成，包含散件和 2-3 件套效果 |
| 传说 | 橙 | 高级加成，包含 2-5 件套效果 |
| 神话 | 红 | 顶级加成，包含 2-7 件套效果 |

## 10. 套装效果

紫色及以上品质可以出现套装效果。

| 品质 | 套装范围 |
| --- | --- |
| 史诗 | 2-3 件套 |
| 传说 | 2-5 件套 |
| 神话 | 2-7 件套 |

套装效果方向：

| 类型 | 示例 |
| --- | --- |
| 经验加成 | EXP 获取 +10% |
| 移动加成 | 移动速度 +8% |
| 生命加成 | 最大生命 +12% |
| 伤害加成 | 最终伤害 +8% |
| 校园币加成 | 校园币获取 +15% |
| 学分对抗 | 学分倍率下限提高 |

学分对抗类套装可以强化本作特色，例如：

```text
2 件套：学分不足时，玩家伤害倍率下限由 30% 提高至 40%。
4 件套：受到 Boss 学分技能伤害上限由 200% 降低至 180%。
```

## 11. 掉落规则

精英、稀有精英和 Boss 都可以掉落增益装备。难度越高，越容易掉落高品质装备。

| 来源 | 掉落方向 |
| --- | --- |
| 普通怪 | 材料、校园币，低概率普通/优秀增益装备 |
| 精英 | 材料、校园币，较高概率优秀/精良增益装备 |
| 稀有精英 | 精良装备为主，有概率史诗装备 |
| Boss | 必定掉落增益装备，较高概率史诗，低概率传说或神话 |
| 宝箱 | 消耗校园币开启，按章节等级掉落对应品质 |

掉落遵循两个原则：

```text
章节越后，装备品质上限越高。
敌人难度越高，高品质掉率越高。
```

## 12. 章节推进原则

章节推进采用软门槛：

- 学分越高，挑战 Boss 越稳定。
- 学分不足，也允许挑战 Boss。
- 如果玩家低学分击败 Boss，仍可进入下一章节。
- 学分主要影响 Boss 和部分稀有敌人的攻防倍率，不直接阻断主线。

这样可以保留成长目标，也允许玩家通过装备、暴击、等级和操作策略完成越级挑战。

## 13. 当前可定稿规则

当前基础框架可定为：

1. 学分是章节挑战的软门槛，不是绝对通关门槛。
2. 学分影响带有学分防御的敌人，主要是稀有精英、Boss 和特殊怪。
3. 普通怪和多数普通精英保持简单扣血，不受学分影响。
4. Boss 可拥有主动学分技能，玩家学分不足时受到更高伤害。
5. 物理暴击为 150%，魔法暴击为 200%。
6. 校园币用于开宝箱、购买道具、出售道具和后续合成。
7. 道具分为材料、任务道具、增益装备。
8. 增益装备不改变外观，只提供数值或特殊效果。
9. 装备栏限制 10 格。
10. 增益装备按白、绿、蓝、紫、橙、红六档品质区分。
11. 紫色以上装备可以拥有套装效果。
12. 精英、稀有精英和 Boss 均可掉落增益装备，难度越高，高品质掉率越高。
