发布日期：2026-07-12

部署方式
1. 先备份线上网站目录与 play-data.sqlite3 数据库。
2. 解压全量包，将包内文件上传到网站根目录并覆盖同名文件。
3. 不要删除或覆盖服务器现有的 play-data.sqlite3；全量包不包含数据库。
4. 保持 assets、vendor 目录结构不变，并保留服务器对 mp4 的 HTTP Range 支持。
5. 清理 CDN/浏览器缓存后检查 /play.html、/index.html 与 /opc-console.html。
