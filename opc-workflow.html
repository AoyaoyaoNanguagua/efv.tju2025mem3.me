增长闭环 Growth Loop

一人公司通过 Agent 网络获得持续生产、宣发和复盘能力。An OPC gains production, campaign and feedback capacity through Agents.

主理人保留方向和质量判断，Agent 承担可规模化的生产与分发任务，合作伙伴可以在内容、测试、渠道和商业判断上贡献价值。

1. 主理人定向 Owner DirectionIP 命题、审美、预算、风险边界、产品里程碑和验收标准。
2. 协议设计 Protocol Design任务契约、提示词规则、文件边界、素材规格、API 形态和测试关卡。
3. Agent 生产 Agent Production策划、叙事、概念图、精灵、地图、代码、音频、测试和文档。
4. 版本整合 Integration Build合并资产、连接系统、检查质量，形成可展示、可试玩、可传播的网页版本。
5. 体验评估 Experience Review主理人与协作者评估手感、可读性、趣味性、公共质量和品牌契合度。
6. AI 宣发 AI Campaign生成预告片、短视频、封面、文案变体和发布包。
7. Agent 分发 Agent Distribution在主理人监督下，为小红书等平台准备差异化发布队列。
8. 市场与财务 Market & Finance流量、留存、评论、转化、成本、营收信号和产品组合决策回到主理人。

Agent 系统 Agent System

每个 Agent 都服务于明确的公司资产目标。Each Agent serves a clear company asset goal.

资产目标让内容、美术、代码、测试、宣发和财务分析可以并行推进，并在主理人审阅后进入下一轮增长。
