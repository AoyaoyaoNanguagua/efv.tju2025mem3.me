公司定位 Company Thesis

OPC 经营的不是一款游戏，而是一组持续增值的 IP 资产。The OPC operates a compounding portfolio of IP assets, not a single game.

游戏只是 IP 价值的一种载体。玩法、章节、活动、视频、实体周边、授权与社群关系都会进入同一套资产账本，接受持续投资、验证、扩张或止损。

01IP 资产运营管理原创世界、角色、故事和品牌识别，形成长期可复用资产。
02AI 生产系统策划、代码、美术、测试、部署与内容 Agent 按工作订单执行。
03组合投资决策把 Token、注意力、现金与渠道配置到回报最高的资产线。
04市场反馈系统玩家、客户、平台与营收信号持续回流下一轮决策。

每日循环 Daily Operating Cycle

每天从信号开始，以新的投资判断结束。Every day begins with signals and ends with a new capital decision.

定时任务只负责唤醒相应角色。订单状态、实际消耗、交付证据和人工审批全部沉淀在中枢，不依赖某个 Agent 永久在线。

08:00数据汇总市场、玩家、产品与财务信号进入账本。
08:30组合决策决策 Agent 提交今日配置与停止条件。
09:00人类批准主理人确认方向、预算和高风险动作。
09:10订单生成中枢按角色、优先级和依赖创建队列。
DAY生产与 QA工作节点领取、执行、测试并提交证据。
RELEASE部署与宣发通过验收的资产进入发布与市场节点。
23:00财务复盘消耗、收入和客户信号返回下一轮决策。

工作订单 Work Order

Agent 领取的不是一句提示词，而是一份可验收的业务订单。Agents claim reviewable business orders, not loose prompts.

订单把资本决策转化为边界清楚、可暂停、可重试、可审计的执行单元，也让人类可以在任何阶段介入。
