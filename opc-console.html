协议蓝图 Protocol Blueprint

每一次运行都从业务目标进入协议包，再进入 Agent 和工具调用。Each run starts from a business brief and moves through protocol, agents and tools.

协议包定义输入、产出、文件边界、验收口径、预算记录和发布渠道，让不同 Agent 可以并行工作，并让投资人与合作方看到可扩展的生产机制。

Owner BriefIP 目标、受众、预算、里程碑、风险边界。
Protocol Pack任务拆解、提示词模板、文件契约、素材规格。
Agent RunCodex 类 Agent 调用文件、浏览器、图像、视频、数据工具。
Review Gate主理人和协作者检查体验、质量、品牌和发布风险。
Release Ledger版本资产、宣发内容、成本记录、平台信号和复盘结论。

任务队列 Mission Queue

控制台把公司增长拆成可执行任务。The console turns company growth into executable missions.

每个任务都对应可交付资产、验收标准和反馈数据，方便主理人、协作者和顾问围绕同一套事实做判断。

宣传视频包 Campaign Video Pack 生成预告片脚本、封面帧、短视频切条、平台标题和发布时间表。 Video Agent · Copy Agent · Distribution Agent
多人同步回归 Online Sync QA 浏览器打开试玩流程，检查 WebSocket 同步、角色状态、Boss 行为和移动端体验。 QA Agent · Browser Tool · Server Log

宣发财务复盘 Campaign Finance Review 汇总平台表现、互动质量、制作成本、工具消耗、潜在合作和下一轮预算建议。 Insight Agent · Finance Agent · Market Ledger
